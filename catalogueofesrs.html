Here is a page of both official and unofficial ESRs. I will list the more notable (and interesting) ones (at least with alphabets I can easily decipher); while I don't hate using neography or completely new glyphs to write English, there is way too many of them. If you want to check all of them out, just go to this Omniglot page, or go to r/neography or r/conorthography.

2. Changes to the Latin Alphabet

This is defined as large changes in spelling (often following orthographic rules of other languages as well), though still limited to the Latin Alphabet (diacritics allowed).

Simpel-Fonetik: Finnish

SaypYu (it does add a schwa)

Regularized Inglish


3. Latinate spelling patches/tweaks

This is defined as minor changes to spelling; using diacritics as a reading aid instead of part of the formal orthography also falls into this category, since it does not fundamentally change English spelling.

The various pronunciation re-spellings

SoundSpel

Cut Spelling

SR1

Traditional Spelling Revised


4. Using other alphabets to write English

The reason why I don't put it with the same category as new alphabets is because many use existing sets of letters (e.g. Greek, Cyrillic, Devaganari) to write English.

(And, perhaps as a result, those are never published as official ESRs lol. Just a fun design challenge.)

Anglika: Greek

Hellerican: Greek

Cyrillisch: Cyrillic

Cyrenglish: Cyrillic

Renglish: Cyrillic

Eigana: Japanese

Katakana Eigo

Neo-Gothic: Gothic (wow)

Krostt: a mix of Latin, Greek and Cyrillic
