What is Cooked English?

Cooked English (CE; *Cóked Énglisç in CE) is a system I designed as a fun little project/challenge to fix English spelling.

Goals:

1. Make English spelling more consistent

2. Trace back etymology when we need it

3. Make English follow more of the alphabetic principle (though with the above restrictions, CE can't be fully phonemic; there are, however, sets of rules to dictate pronunciation)


The writing system

1. New letters

Þ: thorn (þorn). This letter originated from a rune of the same name, and was used in Old English (alongside Ð, eth) to spell the sounds we now spell as "th": the voiceless and voiced dental fricatives.

Ç: çe (che). This letter originated as a Visigothic Z, and is used in e.g. French to denote a /s/ before back vowels, and e.g. Turkish to denote /tʃ/. This letter will be used in the same way as in Turkish to write "ch".

Ᵹ: ᵹe (je). This letter originated as a variant of writing the letter "g" (also known as insular g). Here it is used to write the /dʒ/ sound, commonly represented by "g", "dg", and "j".

Ȝ: joȝ (yogh). This letter started off as the above one, but simplified in writing. Here it represents a remnant of the old "gh" that has gone silent (in fact, this letter is also used to write that sound in Middle English), and to transcribe the "ch" in Scottish "loch".

Æ: asç (ash). Obviously, this letter is an "A" and an "E" smashed together. This letter will be used like how it is used in Danish and Norwegian, to denote the /ɛ/ sound when it is written "ea".

Ø: øþel (erthel). Now I know that this letter commonly represents /ø/ in the languages that use it, but it is similar enough to English /ɝ/ that I will use it to write "or" when it is pronounced that way. Additionally (though optional) I will also denote o = /ʌ/ this way.

For Ȝ and Ᵹ, they are interchangeable with G under some words, e.g. síȝn (sign) but signature.

Optionally:

Ŋ can be used as an abbreviation for "ng".

Ð can be used as an alternative to Þ.


2. Consonant use changes

J will solely be used for /j/, with no exceptions.

G will solely be used for /g/; there's no such thing as a soft "g" anymore.

Hard and soft C will be kept; though made a tiny bit more consistent.


3. The vowel system

a. The schwa

How do we write the schwa? My solution is: just leave it there, mostly.

the letter Ø comes into play as notation for a reduced "o", leaving mostly A and E as schwas instead.

d. Other vowel changes

ei which represents /e, ə, ɪ/ is changed to "e" or "i".

ai representing /ɛ/ is spelt æ.

façade is spelt "facêad", and limaçon is spelt "limacêon".

ocean is spelt "ôsçêan" to better match with "ôsçéanic" (oceanic).
