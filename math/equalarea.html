2 triangles of equal area: how to solve sides?

back to interchange    main page


areas of triangles

Someone once presented me with a question:

A triangle has sides of lengths 29, 29, and 40. Find another isosceles triangle with the same perimeter and area that also has sides of integral lengths.

Additionally, he has requested me to find the general solution to this type of problem, where we have an isosceles triangle of base 2x and height y.

Interestingly enough, this delves into some crazy relations with cubics, the difference of squares theorem, and sine rules.


solving the original problem

1. Splitting and setting variable

We split the original triangle in half down the 40 side, giving us 2 triangles of [20, 21, 29]. (21 is derived from the Pythagorean theorem, aka the height.)

The area of the entire triangle is 420 (units sq.), and the perimeter is 98.

Suppose we have another isosceles triangles of that same area and perimeter. That means we can divide it into 2 right-angled triangles too.

In that case, let the side lengths of each right-angled triangle be a, b and √(a^2 + b^2) (without loss of generality, we let 2a be one of the sides of the larger isosceles triangle, aka it has side lengths 2a, √(a^2 + b^2), √(a^2 + b^2))

2. Seeting up equations

We have:

ab = 420, re-writing it as a = 420/b (eq1)

2a + 2√(a^2 + b^2) = 98, aka a + √(a^2 + b^2) = 49 (eq2)

Making a the subject:

√(a^2 + b^2) = 49 - a

a^2 + b^2 = a^2 - 98a + 2401

b^2 = 2401 - 98a

3. The math

Subbing a = 420/b:

b^2 = 2401 - 98(420/b)

b^3 = 2401b - 41160

b^3 - 2401b + 41160 = 0

Resolving the cubic, we get at b = -56 (rejected), b = 21 (repeat of original), and b = 35. When b = 35, a = 12.

Hence we have triangle with sides 24cm, 37cm, 37cm


factorization of the cubic

The good thing here is, we know that b = 21 (i.e. a = 20) is one of the solutions of the above cubic.

Hence, we can factorize the polynomial without going through the need of cubic solving:

b^3 - 2401b + 41160 = 0

(b - 21)(b^2 + 21b - 1960) = 0

(b - 21)(b - 35)(b + 56) = 0

b = 21, b = 35, b = -56


the general case

For an isosceles triangle of base 2x cm and height y cm:

1. Setting variables

Let A = area, so A = xy; let P = perimeter, so P = 2x + 2√(x^2 + y^2)

Suppose we have another triangle [2m, √(m^2 + n^2), √(m^2 + n^2)]; m and n are not equal to both x and y.

Then we have A = mn and P = 2m + 2√(m^2 + n^2) as well.

2. Equations

From the above, we have n = A/m and 2√(m^2 + n^2) = P - 2m

Rewriting a bit:

4(m^2 + n^2) = P^2 - 4Pm + 4m^2

4(A/m)^2 = P^2 - 4Pm

4Pm^3 - (P^2)m^2 + 4A^2 = 0, cubic in terms of m

3. Solving the cubic

Since (x, y) (i.e. m = x) is a solution, we can simplify the cubic directly.

Alternately, one can derive a cubic in terms of n and use n = y to simplify it as well.


another example

Suppose we have x = 33, y = 56 (√(x^2 + y^2) = √(33^2 + 56^2) = 65.) This gets us a triangle of [66, 65, 65] (very close to being equilateral, lol.), where A = 33x56 = 1848, P = 66 + 65 + 65 = 196.

Making the substitutions:

4(196)(m^3) - (196)^2(m^2) + 4(1848)^2 = 0

196m^3 - 9604m^2 + 1848^2 = 0

m^3 - 49m^2 + 17424 = 0

Since we know that m = 33 is a root, we can directly simplify it as (m - 33)(m^2 - 16m - 528) = 0

However, m = 33 is the only integral root, meaning that for this particular area and perimeter, (66, 65, 65) is the only isosceles triangle with integral side lengths.


a special case of irrational area and perimeter

In fact, y can actually be a non-integer, since y isn't a side in the original isosceles.

e.g. for x = 2 and y = 3√13 (√(x^2 + y^2) = 11):

A = 2(3√13) = 6√13, and P = 4 + 11 + 11 = 26

Suppose another triangle 2m, √(m^2 + n^2), √(m^2 + n^2) has the same area and perimeter as above.

Equation time

4(26)m^3 - (26^2)m^2 + 4(6√13)^2 = 0

104m^3 - 676m^2 + 1872 = 0

2m^3 - 13m^2 + 36 = 0

Since m = 2 is a root, this is simplified to (m - 2)(2m^2 - 9m - 18) = 0, giving us (m - 2)(2m + 3)(m - 6) = 0

We get m = 2 (original), m = -1.5 (rejected), or m = 6.

In the case of m = 6, n = √13, and √(m^2 + n^2) = √(36 + 13) = 7. Hence [7, 7, 12] triangle actually has the same area and perimeter as a [11, 11, 4] triangle.


generation of differences of squares and pythagorean triples

Using the above example, suppose the angle between the 2 sides of 7 is θ, and the angle between the 2 sides of 11 is φ

By the sine area theorem, we know that A = 49sinθ/2 = 121sinφ/2

Also, cosθ = (49 + 49 - 144)/(2*7*7) = -23/49 and cosφ = (121 + 121 - 16)/(2*11*11) = 113/121

Consequently, 1 - (cosθ)^2 = 1 - (-23/49)^2 = 1872/2401, and 1 - (cosφ)^2 = 1 - (113/121)^2 = 1872/14641. Notice how 1872 is the numerator of both fractions. In fact:

A^2 = 2401((sinθ)^2)/4 = 2401(1 - (cosθ)^2)/4 = 2401(1872)/(4)(2401) = 468

At the same time, A^2 = 14641((sinφ)^2)/4 = 14641(1 - (cosφ)^2)/4 = 14641(1872)/(4)(14641) = 468

Hence, this particular set of triangles actually also shows that the number 1872 can be expressed in (at least) 2 different ways of the differences of squares, namely 1872 = 49^2 - 23^2 = 121^2 - 113^2

In the case of a square number, we will end up generating a Pythagorean triple.
