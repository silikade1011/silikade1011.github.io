2 triangles of equal area: how to solve sides?

back to interchange    main page


areas of triangles

Someone once presented me with a question:

A triangle has sides of lengths 29, 29, and 40. Find another isosceles triangle with the same perimeter and area that also has sides of integral lengths.

Additionally, I was requested to find the general solution to this type of problem, with an isosceles triangle of base 2x and height y.

Interestingly enough, this delves into some crazy relations with cubics, the difference of squares theorem, and sine rules.


solving the original problem

1. Splitting and setting variable

We split the original triangle in half down the 40 side, giving us 2 triangles of [20, 21, 29]. (21 is derived from the Pythagorean theorem, aka the height.)

The area (A) of the whole triangle is 420 (units sq.), and the perimeter (P) is 98.

Suppose we have another isosceles triangles with the same A and P. Naturally we can divide it into 2 right-angled triangles.

Let the side lengths be [a, b, √(a² + b²)] (without loss of generality, we let 2a be the length of a side, aka this triangle has sides [2a, √(a² + b²), √(a² + b²)].)

2. Setting up equations

We have:

ab = 420, aka a = 420/b

2a + 2√(a² + b²) = 98, aka a + √(a² + b²) = 49, which can be re-written:

√(a² + b²) = 49 - a

a² + b² = A² - 98a + 2401

b² = 2401 - 98a

3. The math

Subbing a = 420/b:

b² = 2401 - 98(420/b)

b³ = 2401b - 41160

b³ - 2401b + 41160 = 0

Look, a depressed cubic. Fortunately, we know that b = 21 (i.e. a = 20) is one of its solutions.

Hence, we can factorize the polynomial without going through the need of cubic solving:

(b - 21)(b² + 21b - 1960) = 0

(b - 21)(b - 35)(b + 56) = 0

b = 21 (original), b = 35 (hence a = 12), b = -56 (rejected)

Subbing the other value of b, we get the solution [24, 37, 37].


so what about the general case?

I did promise the requester, so here it is. For an isosceles triangle of base 2x units and height y units:

1. Setting variables

A = xy and P = 2x + 2√(x² + y²).

Suppose we have another triangle [2m, √(m² + n²), √(m² + n²)]. Then we have A = mn and P = 2m + 2√(m² + n²).

2. Equations

From the above, we have n = A/m and 2√(m² + n²) = P - 2m

Rewriting a bit:

4(m² + n²) = P² - 4Pm + 4m²

4(A/m)² = P² - 4Pm

4Pm³ - P²m² + 4A² = 0; a cubic in terms of m.

3. Solving the cubic

Since (x, y) (aka m = x) is a solution, we can simplify the cubic directly.

Alternately, one can derive a cubic in terms of n and use the solution n = y to simplify it as well.


another example

Suppose we have x = 33, y = 56 (√(x² + y²) = √(33² + 56²) = 65.), getting us a triangle of [66, 65, 65], where A = 33x56 = 1848, P = 66 + 65 + 65 = 196.

Making the substitutions:

4(196)m³ - 196²m² + 4(1848²) = 0

196m³ - 9604m² + 1848² = 0

m³ - 49m² + 17424 = 0

(m - 33)(m² - 16m - 528) = 0

m = 33 is the only integer root, meaning that for this particular area and perimeter, [66, 65, 65] is the only isosceles triangle with integer side lengths.


a special case of irrational area and perimeter

Notice how we didn't set y as a side? That means y can be a non-integer.

e.g. for x = 2 and y = 3√13 (aka √(x² + y²) = 11):

A = 2(3√13) = 6√13, and P = 4 + 11 + 11 = 26

Suppose another triangle 2m, √(m² + n²), √(m² + n²) has the same area and perimeter as above.

Equation time

4(26)m³ - 26²m² + 4(6√13)² = 0

104m³ - 676m² + 1872 = 0

2m³ - 13m² + 36 = 0

(m - 2)(2m² - 9m - 18) = 0

(m - 2)(2m + 3)(m - 6) = 0

We get m = 2 (original), m = -1.5 (rejected), or m = 6.

In the case of m = 6, n = √13, and √(m² + n²) = √(36 + 13) = 7. Hence a [7, 7, 12] triangle has the same area and perimeter as a [11, 11, 4] triangle.


extras: generation of differences of squares and pythagorean triples

Using the example directly above, suppose θ is the angle between [7, 7], and φ is the angle between [11, 11].

By the sine area theorem, we know that A = 49sinθ/2 = 121sinφ/2.

Also, cosθ = (49 + 49 - 144)/(2x7x7) = -23/49 and cosφ = (121 + 121 - 16)/(2x11x11) = 113/121.

Consequently, sin²θ = 1 - (-23/49)² = 1872/2401, and sin²φ = 1 - (113/121)² = 1872/14641. Notice how 1872 is the numerator of both fractions. In fact:

A² = 2401sin²θ/4 = 2401(1872)/(4)(2401) = 468. At the same time, A² = 14641sin²φ/4 = 14641(1872)/(4)(14641) = 468.

Hence, we can derive 2 different ways of writing 1872 as difference of squares: 1872 = 49² - 23² = 121² - 113². In the case of a square number, we will end up generating a Pythagorean triple.
